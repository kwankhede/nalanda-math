Key Terminology & Concepts

Like terms: Same variables with same powers ‚Üí can be combined

Operations on Algebraic Expressions

1Ô∏è‚É£ Addition of Expressions

Method: Combine like terms by adding their coefficients.

Example 1: Add (3x + 5) and (2x - 3)
Solution:
= 3x + 5 + 2x - 3
= (3x + 2x) + (5 - 3)
= 5x + 2
Example 2: Add (4a + 3b - 2) and (2a - 5b + 7)
Solution:
= 4a + 3b - 2 + 2a - 5b + 7
= (4a + 2a) + (3b - 5b) + (-2 + 7)
= 6a - 2b + 5

2Ô∏è‚É£ Subtraction of Expressions

Method: Change the sign of every term in the expression being subtracted, then combine like terms.

Example 1: Subtract (2x - 5) from (7x + 3)
Solution:
= (7x + 3) - (2x - 5)
= 7x + 3 - 2x + 5 ‚Üê Change signs
= (7x - 2x) + (3 + 5)
= 5x + 8
Example 2: Subtract (3a + 2b) from (5a - 4b)
Solution:
= (5a - 4b) - (3a + 2b)
= 5a - 4b - 3a - 2b
= (5a - 3a) + (-4b - 2b)
= 2a - 6b
